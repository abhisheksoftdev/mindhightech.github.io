Web Development Course Syllabus

HTML

HTML is a hypertext document markup language. As of this course's creation, HTML 5 is the most up-to-date markup language. HTML uses static tags only. For further information on how to use hyperlinks in HTML documents, see Hypertext Markup Language (HTML). Mark-up language is the names given to the labels used to define the layout of a website page and the elements that make it up. HTML is used to identify the purpose of a web page.

CSS

The look and feel of a web page may be customised using CSS. Color, typeface, and images may all be customised using CSS. Websites may be made more visually appealing and simpler to browse by using CSS. Instead of relying on JS to create animation effects, CSS can do so. CSS separates styling elements like font size, font colour, wire diameter, and the document's background colour for ease of maintenance. As a result, you're free to apply the same design principles to many web pages.

Bootstrap

Developers use Bootstrap to create cutting-edge websites and online apps. This open-source and cost-free solution includes a wide variety of HTML / CSS styles for UI Interface components such as buttons and forms. Bootstrap accepts JavaScript extensions as well. Bootstrap is a web app development framework that is both open and free. Bootstrapping is the process of loading programs into a computer from a smaller beginning programme.

Java Script and Jquery

JavaScript is a text-based language of programming that can be used on both the server and the client-side to make web pages dynamic. The Html markup gives web pages structure and design, while JavaScript injects interaction that attracts website visitors. Adding JavaScript dynamic functionality to a static web page enhances the general user experience. JScript is by far the most efficient and user-friendly scripting language for building dynamic web pages.

JQuery UI

The jQuery JavaScript Library is used as a foundation for this application. The jQuery UI framework was used to build a selected set of user experience interactions, effects, widgets, and themes. jQuery UI is the best choice regardless of whether you're creating complex web applications or just putting date pickers to a frame element.

PHP MySQL

It's a server-side scripting language designed especially for building websites. Because it is run on the website rather than the computer system, PHP is known as a server-side javascript library. MySQL is the most widely used open-source database server (RDBMS). The bulk of database queries in PHP are made using this database system. MySQL was created, distributed, and is supported by the Oracle Corporation. MySQL is a web-based DBMS that runs on a server. MySQL is a great option for projects of any size.
