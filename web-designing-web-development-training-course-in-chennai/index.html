Web Development Course Syllabus

HTML

HTML is a markup language for hypertext documents. When this course was written, HTML 5 was the most current markup language. Only static tags are used in HTML. See Hypertext Markup Language for further information on using hyperlinks in HTML texts (HTML). There are a variety of terms used to refer to labels that describe the structure of a website page and the components that make it up. Using HTML, a web page's purpose may be determined.

CSS

A web page's appearance may be altered using CSS. CSS may be used to change the colour, typography, and picture of a webpage. CSS may be used to make websites more aesthetically attractive and easier to navigate. CSS may be used instead of JS to provide animation effects. Font size, colour, wire diameter, and the document's background colour are all separated in CSS for easier maintenance. There's no limit on how many web pages you may build in the same way.

Bootstrap

When it comes to creating cutting-edge sites and online applications, Bootstrap is used. HTML/CSS styles for UI Ui elements like buttons and forms are included in this open source and freely solution. JavaScript extensions are also supported by Bootstrap. In addition to being open source, Bootstrap is also free. It is the process of putting programmes into a system from a lower starting point.

Java Script

Text-based programming language JavaScript could be used on the client and the server to start making dynamic websites. Architecture and design are provided by Html, but JavaScript adds interactivity that draws web visitors. User experience is improved when JavaScript dynamic capability is added to an otherwise static page. For creating dynamic web pages, JScript is the most efficient and user-friendly scripting language.

This includes JQuery and its user interface (JQuery UI)

The JavaScript-based application is built on the framework of jQuery. User experience actions, animations, gadgets, and themes are all built using the jQuery UI framework. Even if you're only adding date pickers in a frame element, jQuery UI is the ideal option for you.

PHP MySQL

Server-side script meant for developing websites is what it is. PHP is referred to be a server-side javascript library since it is executed on the site instead of the computer system. The most popular open-source database server is MySQL (RDBMS). This database management system is used for the majority of PHP database queries.
