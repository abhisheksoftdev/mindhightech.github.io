HTML

HTML is a markup language for hypertext documents. All mark-up systems have their roots in SGML (the Standard Generalized Mark-up Language). HTML 5 is the most recent markup language, and it's covered in this course. All tags in HTML are static. HTML stands for Hypertext Markup Language and refers to the hyperlinks included inside an HTML page. The labels used to describe page setup and web page components are referred to as mark-up languages. The structure of a web page is defined using HTML.

CSS

A style sheet, often known to what determines how a web page looks and feels. CSS is used to design a web page's appearance and feel. CSS allows us to style HTML components using colour, font, and pictures. CSS is used in web development to make websites more aesthetically pleasing and easier to navigate. CSS, on the other hand, has the capability of implementing animation effects. Styling such as font size, font colour, wire diameter, and the colour is separated using CSS to make document maintenance easier. Because of this, you may use the same style guidelines across numerous web pages.

Bootstrap

In order to build contemporary websites and online applications, developers utilise Bootstrap. There are many Html / CSS styles for UI UI components like buttons and forms in this accessible and free-to-use system. Extensions written in JavaScript are supported by Bootstrap as well. Free and open-source, Bootstrap is a web and web and mobile app development platform. To use a much smaller starting programme to load the required software into a computer is referred to as bootstrapping.

Java Script and Jquery

In order to make websites interactive, JavaScript uses a text-based programming language that may be utilised on the client and the server sides. Html provides websites structure and style; JavaScript, on the other hand, adds interactivity that appeals to users. Making a static web page dynamic with JavaScript enhances the web page's overall user experience. When it comes to creating dynamic web pages, JScript is hands-down the most efficient and user-friendly solution.

JQuery UI

It is developed on the basis of the jQuery JavaScript Library. It is a curated collection of user experience interaction, impacts, widgets, and themes that are created with the help of the jQuery UI library. In any case, whether you're developing highly dynamic online apps or just adding date pickers to a frame element, jQuery UI is the right option. In addition to being open source and free software, jQuery UI is also provided by the jQuery Foundation underneath the MIT License.

PHP MySQL

PHP is a server-side software program that was created specifically for the purpose of developing websites. PHP is considered to be a server-side javascript framework since it is executed on the server rather than on the client's computer. Mysql is the most popular management system that is free and open-source software (free software) (RDBMS). In PHP, it is the most widely used database system, accounting for a majority of all database requests. The Oracle Corporation is responsible for the development, distribution, and support of MySQL. MySQL is a database management system that is installed on a server and operates over the Internet. In both small and big projects, MySQL is an excellent choice.
