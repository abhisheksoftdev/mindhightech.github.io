About Perl:

  • A dynamic programming language developed by Larry wall as a general purpose UNIX scripting language to ease the processing of reports.
  • First version of Perl (Perl 1.0) was released on December 1987 and the latest version (Perl 5.27.7) was released on December 2017.
  • However, Perl has evolved so much and now the Perl family contains two different languages called Perl 5 and Perl 6.
  • Original Perl (Perl 5) was used in various domains like CGI (Computer Generated Imagery), System administration, network programming, finance and bioinformatics.
  • Perl was wittily called as “Swiss Army chainsaw of scripting languages” because of its command, flexibility and importantly, inelegance.
  • Perl supports Object Oriented Programming (OOP), first class functions, data typing and complex data structures.
Perl is available for all major operating systems, for example, as MacPerl for macOS.
